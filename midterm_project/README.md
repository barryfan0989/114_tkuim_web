# 售票查詢系統

組員：413637249游雅萱, 413637454范植翔

## 專案簡介
本專案為一個簡易的校園活動報名系統，使用者可在網頁上填寫報名表單並線上送出，系統包含即時表單驗證與送出確認，並避免重複送出資料。

## 使用技術
- HTML
- CSS
- Bootstrap
- JavaScript
- Constraint Validation API（表單驗證）

## 功能特色
1. 表單即時驗證：使用 Constraint Validation API 與自訂驗證邏輯，於使用者輸入時顯示驗證回饋。
2. 報名資料顯示：送出後於頁面上顯示或列出使用者所填寫的報名資料（示範用途）。
3. 防止重複送出：送出後會停用送出按鈕或採用其他機制以避免使用者重複按下送出。

## 如何本地檢視
1. 在本專案資料夾中打開 `index.html`。
2. 使用瀏覽器（例如 Chrome、Edge）開啟 `index.html`，即可在本地測試表單與互動行為。

## 檔案摘要
- `index.html`：主要頁面與報名表單。
- `script.js`：表單驗證與送出的 JavaScript 邏輯。
- `style.css`：樣式檔案（含 Bootstrap 覆寫或自訂樣式）。
